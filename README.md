# tasmo

## Description
List the tasks you want to work on.

## Environment
- Ubuntu 18.04 or 20.04
- ruby 2.7
- gtk3

## Installation
### ruby
```
$sudo apt install ruby
$sudo apt install ruby-dev
```

### gtk
```
$sudo apt install libgtk-3-dev
$gem install rake
$gem install gtk3
```

## Run
```
$ruby main.rb
```
